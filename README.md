# Practice .NET Application

## Overview
This project is a practice application built using ASP.NET Core, focusing on implementing the base repository pattern and Data Transfer Objects (DTOs). It also includes authentication features using ASP.NET Identity.

## Features
- Practice with ASP.NET Core MVC architecture.
- Implementation of the base repository pattern for data access.
- Usage of DTOs for transferring data between layers.
- Integration of ASP.NET Identity for user authentication.
